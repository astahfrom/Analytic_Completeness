# Formalised completeness of Peter Andrews' Higher-Order Logic in Isabelle/HOL

This folder contains the mechanized proofs described in our submission.

They can be opened in Isabelle 2025-2 with the Archive of Formal Proofs installed.
* Isabelle 2025-2 download: https://isabelle.in.tum.de/website-Isabelle2025-2/
* Archive of Formal Proofs download: https://isa-afp.org/download/
* Archive of Formal Proofs installation: https://isa-afp.org/help/
