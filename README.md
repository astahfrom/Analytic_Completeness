This repository used to hold the [Abstract Consistency Properties](https://isa-afp.org/entries/Abstract_Consistency_Property.html) framework, which is now in the Archive of Formal Proofs.

It now holds a completeness proof for Andrews' Q0 based on the [Metatheory of Q0](https://isa-afp.org/entries/Q0_Metatheory.html) AFP entry by Javier Díaz.
